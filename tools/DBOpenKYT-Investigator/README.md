# DBOpenKYT Investigator

**开源轻量级 Crypto KYT 调查工具**  
**作者**：Xiaobo Dai (DB Airon) - C3O Certified

### 项目定位
专为中小 VASP、Web3 项目方、合规转型者设计的 **Chainalysis Reactor 平民版**。

聚焦真实痛点：**KYT 警报触发后快速调查**，5-10 分钟内完成资金流追踪、风险评估和报告生成。

### 核心功能（MVP）
- 输入地址或交易哈希 → 自动追踪多跳资金流
- 生成清晰的 Mermaid 资金流可视化图（类似 Reactor）
- 自动风险标注（交易所、Mixer、制裁、高风险行为）
- 一键生成专业合规调查报告（Markdown 可转 PDF）
- 支持 Ethereum（后续扩展 BSC、Solana）

### 适用人群
- Web3 Compliance 分析师（币安/OKX 等交易所日常工作模拟）
- 合规转型者构建 Portfolio
- 中小型项目自建轻量 KYT 能力
